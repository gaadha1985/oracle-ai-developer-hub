# Session agent-memory

## Topic
- [[Agent Memory]]

## Evidence
- **[[Episodic Memory]]** — semantic memory doesn't change per iteration but episodic has an effect e.g if a user faces an error that will want to be recorded for next time
- **[[Procedural Memory]]** — Episodic memory helps an agent remember what happened on a prior run and use that history next time.
- **[[Semantic Memory]]** — Procedural memory is the memory of skills and repeated methods, like how to authenticate, call tools, and execute a workflow.
