# Session agent-loop

## Topic
- [[Agent Loop]]

## Evidence
- **[[Memory Update]]** — Memory update is deciding what should persist after the loop so the agent improves over time.
- **[[Evaluation]]** — Evaluation checks whether the result really matched the goal rather than just whether the tool returned success.
- **[[Execution]]** — Execution is the act of changing the environment, calling a tool, or making something happen.
- **[[Tool Selection]]** — Tool selection is deciding which capability to use, while execution is actually running it.
- **[[Context Retrieval]]** — Context retrieval is the step where the agent pulls in relevant memory or information for the current turn.
